# 📧 Email Spam Detector (Machine Learning Project)

A simple and effective Email/SMS Spam Detection system built using **Machine Learning (Naive Bayes)** and deployed with a **Flask web application**.

---

## 🚀 Features

- Detects whether a message is **Spam 🚨** or **Not Spam ✅**
- Uses **TF-IDF Vectorization** for text processing
- Machine Learning model: **Multinomial Naive Bayes**
- Clean and modern **Light Blue UI**
- Real-time prediction via web interface

---

## 🧠 Tech Stack

- Python 🐍
- Scikit-learn
- Pandas
- Flask
- HTML & CSS
